# pollsApi - API for Polls system.

## **Deploy.**
1 Download project
```
git clone url-of-repo
```
2 Install docker, docker-compose
```
https://docs.docker.com/get-docker/
```
3 Run command
```
docker-compose up --build
```
4 Enjoy

## **Dependencies.**

1. Django
2. djangorestframework
3. psycopg2
